Yii 2 Advanced Project Template
===============================

## Installation

1. Clone repository

```
git clone https://github.com/kalny/yii2-template.git project
cd project
```

2. Update vendors

```
composer update
```

3. Init yii2 advanced application

```
php init
```

4. Configure the database in file ```@common/config/main-local.php```

5. Apply migrations

```
php yii migrate
```
